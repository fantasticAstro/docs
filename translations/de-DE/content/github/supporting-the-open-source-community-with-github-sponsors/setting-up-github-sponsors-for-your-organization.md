### Deine Bankinformationen einreichen

Als unterstützte Organisation musst Du Auszahlungen auf ein dediziertes Bankkonto Deiner Organisation in einer unterstützten Region erhalten. Du kannst ein Unternehmensbankkonto über Dienste wie [Open Collective](https://opencollective.com/) und [Stripe Atlas](https://stripe.com/atlas) erhalten. {{ site.data.reusables.sponsors.stripe-supported-regions }}
